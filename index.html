我的软件使用笔记

VS Code 使用

1. 打开软件：双击图标启动

2. 新建文件：Ctrl + N

3. 保存文件：Ctrl + S

4. 自动保存：设置 -> auto save 打开

LaTeX 配置

1. 安装 MikTeX 或 Tex Live

2. VS Code 安装 LaTeX Workshop 插件

3. 关闭自动编译：autoBuild: never

4. 手动编译：点击右上角绿色按钮

GitHub 部署网页

1. 新建仓库：用户名.github.io

2. 上传 index.html

3. Settings -> Pages 开启

4. 等待1分钟即可访问
